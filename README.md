# reQUnit [![Build Status](https://secure.travis-ci.org/azu/reQUnit.png?branch=master)](http://travis-ci.org/azu/reQUnit)

Converting QUnit to jasmine.

## Getting Started

```js
$ node bin/reQUnit.js example/simple-code.js
```
